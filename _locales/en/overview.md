Every time you open a new window or tab, you'll be greeted by a clean page designed for Microsoft users and admins. Yako
helps you quickly navigate to the Microsoft portals you use every day.

Yako was built as a focused new tab replacement for anyone who uses Microsoft 365, Azure, Entra, Intune, and other
Microsoft services. It puts the portals you need front and center, with zero clutter.

Key features: Fast & lightweight! Search bar with instant suggestions for 200+ Microsoft portals Quick Links to your
most-used portals Clock with analog and digital modes Greeting with your name Dark mode Solid color or local image
backgrounds Notes widget Custom CSS styling Privacy focused - only storage permission required, no data collection, no
accounts, no external API calls

Yako aims to look and feel minimal, while giving you fast access to the tools you need. Add Quick Links from a curated
catalog of Microsoft portals, or type in the search bar to instantly find any portal by name or URL.

The project is entirely open source, developer friendly and respects your privacy. We do not collect any data, nor do
you need to create an account to use it. The extension only requires the storage permission.

---

Fast & lightweight! Yako is built with vanilla TypeScript and optimised for the best possible performance.

Search bar The integrated search bar provides instant suggestions from a catalog of 200+ Microsoft portals sourced from
merill/cmd and msportals.io. Find any portal by name, URL, or category.

Quick Links Add your most-used Microsoft portals as Quick Links for one-click access. Browse the full catalog to
discover portals you didn't know existed, or add custom links to any URL.

Clock The default clock is a clean digital display. You can also switch to an analog clock face and customise it to your
liking.

Dark mode Yako includes automatic dark mode that adapts to your system preference.

Privacy focused Yako only requires the storage permission. It makes no external API calls, collects no data, has no
account system, and relies on no third-party services. All portal catalog data is bundled locally with the extension.
Being open source means you can verify these claims yourself.

Custom CSS styling Have a specific look in mind? The custom CSS section lets you fully customise the appearance of your
new tab page.
